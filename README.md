<h1 align="center">Tiled API documentation.</h1>

[View the docs](https://tiledco.github.io/api-docs)

Installation
------------------------------

```shell
npm install
```

Run locally
---------------------------------

```shell
npm start
```

Deploy to Github pages
---------------------------------

```shell
npm run deploy
```
